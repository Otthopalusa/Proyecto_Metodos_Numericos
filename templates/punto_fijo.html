Método de Punto Fijo

Encuentra las raíces de una ecuación utilizando el método de Punto Fijo

Ingresa la ecuación cuya raíz quieres encontrar. Ejemplo: x**2 - 4
Ingresa una función en términos de 'x' donde g(x) = x en el punto fijo. Ejemplo: (x**2 + 4)/2
El punto inicial para comenzar la iteración
El criterio de parada (valor predeterminado: 1e-6)
Número máximo de iteraciones (valor predeterminado: 100)
Consejos para el método de punto fijo:
  • La función g(x) debe ser tal que g(r) = r donde r es la raíz buscada.
  • Para que converja, |g'(x)| < 1 en un entorno de la raíz.
  • Una forma común de obtener g(x) es despejar x de la ecuación original.
  • Elija un valor inicial x0 lo más cercano posible a la raíz estimada.
Ejemplos de funciones que convergen

Ejemplo 1: Para la ecuación x² - 4 = 0 (raíz en x = 2)

  • Función g(x) = sqrt(4) = 2
  • Valor inicial: x₀ = 1

Ejemplo 2: Para la ecuación x³ - x - 2 = 0

  • Función g(x) = (x³ - 2)/x
  • Valor inicial: x₀ = 1.5

Ejemplo 3: Para la ecuación cos(x) - x = 0

  • Función g(x) = cos(x)
  • Valor inicial: x₀ = 0.5
